# Changelog

## 1.0.0 — Initial Release

- 9 slash commands for full job search pipeline
- Multi-agent parallel search with configurable agent count
- 6-dimension scoring rubric with ATS platform detection
- Per-job application packages (tailored resume, cover letter, instructions)
- Interview preparation with company research, predicted questions, and STAR-format answers
- Interactive mock interviews with real-time scoring (standard, tough, gauntlet difficulty)
- Post-interview debrief with interviewer intent analysis
- AI & modern tooling positioning section in interview prep
- Multi-candidate support with isolated workspaces
- Unemployment reporting data generation with company address lookup

## Roadmap

See [open issues](../../issues) for planned improvements. Key areas:
- Configurable data root across all commands
- Source selection for targeted searches
- Scoring rubric improvements for startup-heavy searches
- Agent architecture optimization (search/score split from generation)
- Interview prep export for printable reference cards
